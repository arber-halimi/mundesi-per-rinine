# Mundesi per Rinine

Platforme web per te ndihmuar te rinjte e Kosoves dhe diaspores te gjejne bursa, pune, praktika, trajnime, gara, shkembime dhe mundesi tjera zhvillimi.

## Status

Projekti eshte ne fazen fillestare te foundation setup.

## Tech Stack i planifikuar

- ASP.NET Core Web API
- Entity Framework Core
- ASP.NET Core Identity
- PostgreSQL
- React
- TypeScript
- Vite
- TanStack Query
- React Hook Form
- Zod

- .NET SDK 10
- Node.js

- Docker Desktop, later for local PostgreSQL
